# 🚨 CrisisSense AI

> **An AI-powered, action-first emergency response system designed to save time and lives.**

---

## 🧠 Problem Statement

In emergency situations, people panic and are often unable to explain their condition or take complex actions.
This delay in communication reduces response time and can lead to serious consequences.

Additionally, bystanders who witness emergencies often want to help but don’t have a fast and effective way to act.

---

## 💡 Solution

**CrisisSense AI** is a mobile application that removes delay by enabling **instant emergency response with a single tap**.

* No forms
* No waiting
* No confusion

The system **acts first** by sending alerts immediately, and then collects additional information if available.

---

## 🚀 Key Features

### 🔴 Action-First Emergency Button

* One tap → instant alert
* No questions asked
* Designed for panic situations

---

### 🧍 Self Emergency Mode

* Used when the user is in danger
* Automatically sends:

  * 📍 Live location
  * ⏱ Timestamp
  * 🚨 Alert notification

---

### 👀 Witness Emergency Mode

* Allows citizens to report accidents or emergencies nearby
* Turns users into **first responders**

---

### 📍 Real-Time Location Sharing

* Captures GPS instantly
* Shares accurate location with responders

---

### 🧠 AI-Powered Urgency Detection

* Voice input analyzed for keywords like:

  * "help", "pain", "accident"
* Classifies urgency level:

  * 🔴 High
  * 🟡 Medium
  * 🟢 Low

---

### 📡 Offline / Low Network Support

* Works even in poor connectivity
* Uses fallback mechanisms (simulated)

---

### 📊 Confirmation System

Displays:

* ✅ Emergency alert sent
* 📍 Location shared
* 👨‍👩‍👧 Contacts notified
* 🚑 Help is on the way

---

## 🧩 System Flow

User taps Emergency
→ Location captured
→ Alert sent instantly
→ Optional details added
→ Confirmation displayed
